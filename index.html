One-Time Database Setup
Run this SQL in your Supabase project to enable withdrawals
STEP 1 — Go to Supabase → SQL Editor → New Query
-- Run this in Supabase SQL Editor CREATE TABLE IF NOT EXISTS withdrawals ( id TEXT PRIMARY KEY, date TEXT NOT NULL, time TEXT NOT NULL, account TEXT NOT NULL, amount NUMERIC(12,2) NOT NULL, note TEXT, created_at TIMESTAMPTZ DEFAULT now() ); -- Allow anon read/write (same as sales table) ALTER TABLE withdrawals ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all_withdrawals" ON withdrawals FOR ALL USING (true) WITH CHECK (true);
STEP 2 — Click RUN, then refresh this app
